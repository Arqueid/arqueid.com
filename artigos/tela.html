Guia de Resolução: Configurar 2 Monitores com Tela do Notebook Queimada

Último vez que tive o problema: usei apenas conectado o monitor vga (como segunda tela), tornei ela tela principal para poder dar video na senha de loginquando reiniciar. e reiniciei enquanto reiniava (ficou preto) lógo apareceu a tela de login, dai conectei o hdmi e deu certo.

Passo 1: Desativar a tela estragada (Modo Cego)

1. Pressione as teclas Windows + P juntas.

2. Pressione a tecla Seta para Baixo 4 vezes seguidas.

3. Pressione a tecla Enter.

Ação: Este comando ativa o modo "Segunda tela somente". O Windows corta o sinal de vídeo da tela do notebook e envia a imagem apenas para as saídas externas.

Passo 2: Acessar a configuração de vídeo

1. Pressione as teclas Windows + I juntas para abrir as Configurações.

2. Clique na opção Sistema.

3. No menu lateral, clique em Tela.

Passo 3: Definir o HDMI como tela principal

1. Clique no botão Identificar. Vai aparecer um número grande em cada monitor. Exemplo: 2 para HDMI e 3 para VGA.

2. Clique no quadrado correspondente ao monitor HDMI (Tela 2) para selecioná-lo.

3. Role a página para baixo até encontrar a seção "Várias telas".

4. Marque a caixa Tornar este meu vídeo principal.

5. Logo acima ou abaixo desta caixa, abra a lista de opções e escolha Estender área de trabalho a esta tela.

Passo 4: Ativar o monitor VGA e organizar a ordem

1. Volte para a parte de cima da página, onde estão os desenhos dos monitores.

2. Clique no quadrado do monitor VGA (Tela 3) para selecioná-lo.

3. Role a página para baixo novamente, vá em "Várias telas" e selecione Estender área de trabalho a esta tela.

4. Volte para o topo da página com o desenho dos monitores.

5. Clique, segure e arraste os quadrados para que a ordem na tela fique igual à posição física dos monitores na sua mesa (Exemplo: 3, 2, 1).

6. Clique no botão Aplicar para salvar a organização.

Pronto. O Windows deixará a configuração salva. A tela do notebook ficará desligada e você usará os dois monitores externos estendidos.
