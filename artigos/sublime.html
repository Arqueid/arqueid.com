Sublime Text Portátil com Autocompletar (Emmet)

Editor de código leve, portátil e pronto para uso.

⬇️ Baixar Sublime Text Portátil

Vantagens da Versão Portátil

O Sublime Text portátil não requer instalação no Windows. Ele pode ser executado diretamente de um pen drive ou salvo em qualquer pasta do seu disco rígido.

Todos os dados de configuração, como cores de fundo (temas) e plugins instalados, ficam armazenados localmente dentro de uma pasta chamada Data. Isso permite manter o seu ambiente de trabalho intacto em qualquer computador.

Plugin Emmet (Autocompletar)

Esta versão já inclui o plugin Emmet previamente configurado. O Emmet é uma ferramenta que acelera a escrita de código HTML e CSS.

Como usar o autocompletar:

1. Crie um novo arquivo e salve com a extensão .html.

2. Digite o sinal de exclamação ! no editor.

3. Pressione a tecla Tab.

A estrutura básica de um documento HTML será gerada instantaneamente. O mesmo atalho (digitar abreviação e apertar Tab) funciona para criar dezenas de tags complexas rapidamente.
