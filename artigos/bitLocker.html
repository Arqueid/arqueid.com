Como Trancar o BitLocker Novamente Sem Desligar o PC

O Windows não possui um botão de "Trancar" para o BitLocker na sua interface normal. Depois de colocar a senha e destrancar um HD ou Pen Drive, a única forma de trancar novamente sem reiniciar o computador é usando o Prompt de Comando (CMD).

Passo 1: Descobrir a letra da unidade

1. Abra o Explorador de Arquivos (Meu Computador / Este Computador).

2. Procure o pen drive, HD ou partição que está com o ícone do cadeado aberto.

3. Anote a letra que aparece entre parênteses ao lado do nome dele (Exemplo: D:, E:, F:).

Passo 2: Abrir o Prompt de Comando como Administrador

1. Clique no botão Iniciar do Windows.

2. Digite cmd no campo de busca.

3. No lado direito do menu, clique em Executar como administrador.

4. Se aparecer uma tela azul de confirmação perguntando se permite alterações, clique em Sim.

Passo 3: Executar o comando de bloqueio

1. Na tela preta que se abriu, digite o comando abaixo:

manage-bde -lock X:

2. Atenção: Apague a letra X e coloque a letra que você anotou no Passo 1. Por exemplo, se o seu pen drive for a letra E, você deve digitar manage-bde -lock E:

3. Aperte a tecla Enter no teclado.

Se der tudo certo, aparecerá uma mensagem confirmando que o volume foi bloqueado. O cadeado voltará a ficar fechado.

Passo 4: Forçar o bloqueio (Caso dê erro)

Se o sistema mostrar uma mensagem dizendo que a unidade está em uso e não pode ser bloqueada, significa que tem algum arquivo aberto puxando dados dela.

1. Feche fotos, vídeos, documentos ou pastas que estejam abertos dentro desse pen drive/HD.

2. Digite o comando abaixo para forçar o fechamento e trancar na marra (substituindo o X pela sua letra):

manage-bde -lock X: -ForceDismount

3. Aperte a tecla Enter.

Pronto. A unidade será ejetada do sistema internamente e trancada com sucesso.
